Packages

Package 1

Pay-per-click
(PPC)


This package will charge advertiser based on the number of clicks their ads receive. This is a popular option for businesses that want to target specific keywords or phrases and only pay for actual engagement with their ads.

Package 2

Flat-rate package
(FRP)

In this package, the advertiser pays a set fee for a certain number of ads to be displayed over a specific period of time. This can be a good option for businesses that want to reach a large audience, but are not as concerned with targeting specific keywords or phrases.

Package 3

Cost-per-action packages
(CPA)


These packages charge advertisers based on the number of specific actions that are taken as a result of their ads. This could include things like sales, sign-ups, or downloads.
